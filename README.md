# 🏥 HealthVista Dashboard

## 📌 Project Overview

The **HealthVista Dashboard** is a Power BI project designed to analyze hospital data and present meaningful healthcare insights through interactive visualizations. It enables users to monitor key metrics, identify trends, and support data-driven decision-making in healthcare management.

---

## 🎯 Objectives

* Analyze hospital and patient-related data.
* Visualize healthcare metrics using Power BI.
* Monitor hospital performance through interactive dashboards.
* Identify trends and patterns in healthcare data.
* Support informed decision-making with data-driven insights.

---

## 🚀 Getting Started

### Prerequisites

* Microsoft Power BI Desktop

### Installation

1. Clone or download this repository.
2. Open `Healthvista dashboard.pbix` in Power BI Desktop.
3. If prompted, reconnect `hospital-dataset.xlsx`.
4. Refresh the data if required.
5. Explore the interactive dashboard.
